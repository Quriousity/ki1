# 설명

a.py
- 4시간봉, 일봉 선택
- 골든크로스 > 롱 앤트
- 데드크로스 > 숏 앤트

b.py
- (input) 종목, 시작, 끝, 봉간격
- (output) 양 거래량 - 음 거래량 = 거래량 합

c.py
- 봉간격
- 모든종목 골든크로스/데드크로스 알림


# 설치
#### 가상환경 설치
```
python -m venv ant
```
혹은
```
python3 -m venv ant
```
#### 가상환경 활성화
```
source ant/bin/activate
```
#### 패키지 설치
```
pip install -r requirements.txt
```

# 실행
```
source ant/bin/activate
```
```
python b.py
```
